# ExamBuilder
ExamBuilder is a secure, intelligent desktop app for teachers to manage questions and generate Word exams. Features: AI text import, one-click .docx export, encrypted storage, and question sharing (.xtk).ExamBuilder 是一款面向教师的安全智能组卷工具，支持题库管理与试卷生成。 核心功能：AI 文本识别导入、一键导出 Word、数据加密存储、题库分享(.xtk)。
